# Cloapedia
Media Berita dengan Framework Laravel

## Instalasi

Menggunakan package manager [composer](https://getcomposer.org/) untuk menginstal Laravel.

```bash
composer install
```

## Migaration 

Buat Database Kemudian jalankan cmd

```bash
php artisan serve
```

## Seed

Isi data Dummy

```bash
php artisan db:seed
```

## Run 

```bash
php artisan serve
```
